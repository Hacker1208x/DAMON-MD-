## ⚙️ Features
- QR Code Generator for WhatsApp Pairing
- Session Sharing System
- Fully Open Source
- Auto QR to DM
- Session ID Generator (DAMON-SESSION-ID)
- Multi Deploy Options (Render, Heroku, Koyeb, etc.)
